# Decisions Ledger — One Row per Phase

> Source of truth for every "Why we chose this approach" block in `full_report.md`, every per-script docstring under `deliverables/scripts/`, and every "Why we chose this approach" markdown cell in `thesis_walkthrough.ipynb`. If a row is empty here, the corresponding block in the deliverables flags it in `CHANGELOG.md` rather than fabricating rationale.

Columns:
- **phase** — which phase or sub-phase
- **choice_made** — the design or implementation decision actually shipped
- **alternatives_considered** — what else was on the table (cited from source docs)
- **decision_criterion** — what made the chosen option win
- **tradeoff_accepted** — what was knowingly given up
- **evidence_path** — where the reader can verify
